# Firewall Rules for Threat Detection Lab
# DO NOT COMMIT ACTUAL IPS OR HOSTNAMES
# Use for reference and documentation of lab network policy

# SIEM to Agents
Allow TCP 514 (syslog) from all servers to SIEM
Allow TCP 9997 (splunk/universal forwarder) from all servers to SIEM
Allow TCP 8089 (SIEM mgmt) from admin workstation only

# Domain Controller
Allow TCP 389 (LDAP) from all lab machines
Allow TCP 636 (LDAPS) from all lab machines
Allow TCP 445 (SMB) from Domain workstations
Allow UDP 53 (DNS) from all lab machines

# Windows Workstation
Allow RDP (TCP 3389) from admin workstation only
Allow WinRM (TCP 5985-5986) from SIEM and DC only

# Linux Server
Allow SSH (TCP 22) from admin workstation only
Allow SSH (TCP 22) from SIEM only
Allow syslog (UDP 514) to SIEM

# General
Deny internet access from lab VMs (isolated network)
Allow internet access from admin workstation only
Deny all inter-VM traffic except as specified above
